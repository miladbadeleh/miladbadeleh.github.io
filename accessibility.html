Accessibility Statement

Last Updated: January 1, 2025

Commitment to Accessibility

Milad Badeleh is committed to ensuring digital accessibility for people with disabilities. I am continually improving the user experience for everyone and applying the relevant accessibility standards to ensure I provide equal access to all users.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. The miladbadeleh.github.io website is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Accessibility Features

I have implemented the following accessibility features on this website:

  • Keyboard Navigation: The website can be navigated using only a keyboard.
  • Text Alternatives: All non-text content has appropriate text alternatives.
  • Contrast Ratio: Text has sufficient contrast against background colors.
  • Resizable Text: Text can be resized up to 200% without loss of content or functionality.
  • Semantic HTML: Proper heading structure and semantic HTML elements are used to enhance screen reader compatibility.
  • Focus Indicators: Visible focus indicators for keyboard navigation.
  • Mobile Responsiveness: The website is fully responsive and usable on mobile devices.

Areas for Improvement

I am aware of the following areas that need improvement to meet full accessibility standards:

  • Some complex visualizations may not have complete text alternatives
  • Some interactive elements may not be fully accessible to screen readers
  • Some form validation messages may not be properly associated with form controls

I am actively working to address these issues in future updates.

Feedback

I welcome your feedback on the accessibility of miladbadeleh.github.io. Please let me know if you encounter accessibility barriers:

  • Email: a.sterling@oxford.edu

I try to respond to feedback within 5 business days.

Technical Specifications

Accessibility of miladbadeleh.github.io relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Assessment Approach

Milad Badeleh assessed the accessibility of miladbadeleh.github.io by the following approaches:

  • Self-evaluation using WAVE Web Accessibility Evaluation Tool
  • Manual testing with screen readers (NVDA, VoiceOver)
  • Keyboard-only navigation testing

Formal Complaints

If you are not satisfied with my response to your feedback, you can contact the relevant authorities in your jurisdiction.
