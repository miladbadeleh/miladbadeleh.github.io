Weld Defects Classification with DenseNet121

Model Overview

This is a deep learning model based on DenseNet121 architecture, fine-tuned for classifying weld aesthetical defects. The model was trained on a specialized dataset of weld images to identify various types of defects with high accuracy.

Training Accuracy
91.01%
Validation Accuracy
90.30%
Test Accuracy
90.48%

Technical Specifications

  • Base Architecture: DenseNet121 (pretrained on ImageNet)
  • Input Size: 300×30 pixels (resized from original images)
  • Input Channels: 3 (RGB)
  • Output Classes: Number of weld defect categories in your dataset
  • Final Layer: Custom linear layer matching the number of defect classes
  • Loss Function: Cross Entropy Loss
  • Optimizer: Adam with learning rate 0.001
  • Training Epochs: 10
  • Batch Size: 32
  • Hardware: CUDA-enabled GPU (model automatically uses GPU if available)

Implementation Details

The model was implemented using PyTorch with the following key components:

Data Preparation

  • Images resized to 300×30 pixels
  • Normalized using ImageNet mean and standard deviation
  • Dataset split into training, validation, and test sets

Model Architecture

The DenseNet121 architecture was used with these modifications:

  • Pretrained weights from ImageNet
  • Final fully connected layer replaced to match the number of weld defect classes

Training Configuration

  • Adam optimizer with learning rate 0.001
  • CrossEntropyLoss as the loss function
  • Batch size of 32
  • Automatic GPU utilization when available

Note: The actual performance may vary slightly when deployed in production depending on the hardware configuration and the similarity of new data to the training dataset.
