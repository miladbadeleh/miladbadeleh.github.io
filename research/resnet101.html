Using ResNet101 for Concrete Crack Detection

This project focuses on leveraging the ResNet101 architecture for detecting cracks in concrete structures. The goal is to improve the accuracy and efficiency of structural health monitoring systems by using deep learning techniques.

Methodology

We utilized a pre-trained ResNet101 model and fine-tuned it on a dataset of concrete crack images. The model was trained to classify images as either containing cracks or being crack-free.

Results

The model achieved an accuracy of 95% on the test set, demonstrating the potential of deep learning for automated crack detection in concrete structures.
